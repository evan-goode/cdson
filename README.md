# cdson

A pure C parser library for the [DSON](https://dogeon.xyz/) data serialization
format, for humans.

## To build

```shell
mkdir build
cd build
meson ..
meson test # optional
sudo meson install
```

## To use

```C
#include <cdson.h>
```

## To understand

Please see documentation inline in cdson.h.
