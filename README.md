# cdson

A pure C parser library for the [DSON](https://dogeon.xyz/) data serialization
format, for humans.

## Building, Installation, and Testing

```shell
mkdir build
cd build
meson ..
ninja
meson test # optional
sudo meson install
```

## Usage

```C
#include <cdson.h>
```

Further documentation is inline in cdson.h.
